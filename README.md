# 🏦 Bank Management System (BAMS)

A full-stack web-based banking management system designed to streamline operations, improve security, and automate core banking processes for banks or SACCOS.

---

## 🚀 Features

- User account management (create, update, delete accounts)  
- Secure authentication & role-based access (Admin, Agent, Customer)  
- Deposits, withdrawals, and transfers between accounts  
- Transaction history with filtering and search  
- Loan application and repayment management  
- Real-time dashboard with key metrics (balances, transactions, loans)  
- Automated confirmation emails and transaction notifications  
- Exception handling (insufficient funds, invalid PIN, recipient not found)  

---

## 🛠️ Tech Stack

### Frontend:
- React.js  
- JSX/HTML  
- CSS  
- Axios for API calls  

### Backend:
- Java (Spring Boot)  
- RESTful APIs  
- JWT authentication & authorization  
- Exception handling and service layer for business logic  

### Database:
- MySQL  

### Tools:
- Git & GitHub  
- Postman (for testing APIs)  
- Maven  

---

## 🌐 System Architecture

- Full-stack web application with separation of concerns: Frontend ↔ Backend ↔ Database  
- Role-based dashboards and endpoints  
- Secure API communication using JWT  
- Service layer for business logic (accounts, transactions, loans)  

---

## 💡 Key Impact

- Automates daily banking operations  
- Improves security with PIN & JWT authentication  
- Tracks transactions accurately and efficiently  
- Enables remote management of accounts and transactions  
- Reduces operational errors and manual work
